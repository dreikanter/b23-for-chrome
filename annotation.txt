Client plugin for b23.ru, URL shortening service.

// Extension annotation for Web Store page (no markup allowed)

Кнопка для сокращения ссылок с помощью сервиса b23.ru. Работает очень просто: первый клик по кнопке 23 сокращает URL открытой в активном табе страницы. Второй клик — копирует ссылку в буфер. В отличии от букмарклета, плагин не требует, чтобы в браузере был установлен Adobe Flash и включена панель закладок. Сокращайте на здоровье.

2012.05.26: Исходники расширения выложены на GitHub, и ссылка Website ведёт теперь как раз на страницу проекта.

Автор будет благодарен тому, кто нарисует приличные промо-картинки для веб-стора размером 440x280, 920x680 и 1400x560 (первую сделал сам, но я не настоящий сварщик и всё такое).
