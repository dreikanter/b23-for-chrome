This is a client plugin for b23.ru web service which makes long URLs shorter.

Плагин позволяет сокращать ссылки с помощью сервиса b23.ru. Все работает элементарно: первый клик по кнопке 23 сокращает URL открытой в активном табе страницы. Второй (опциональный) клик — копирует ссылку в буфер. В отличии от букмарклета, плагин не требует, чтобы в браузере был установлен Adobe Flash.
